parsePageSource(self, _url)

source code 

Metoda sluzacza do parsowania strony o adresie podanym w parametrze. Algorytm dzialania:

  1. Tworzony jest obiekt typu socket
  2. Pobierane jest zrodlo strony o adresie przekazanym w parametrze
    • w razie bledu nastepuje wyjscie z metody
  3. Zrodlo jest czyszczone z pojedynczych tagow
  4. Zrodlo jest parsowane
  5. Tekst wynikowy jest czyszczony ze znacznikow HTML
Parameters:
  • _url (string) - adres URL strony

Requires: urllib
