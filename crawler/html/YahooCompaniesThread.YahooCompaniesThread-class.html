run(self)

source code 

Glowna funkcja watku. Algorytm dzialania:

  1. Pobieranie listy indeksow sektorow
  2. Tworzenie listy spolek
    • pobieranie listy spolek dla kazdego z sektorow
    • dopisywanie pobranych danych do listy ogolnej
  3. Pobieranie podstawowych danych o spolkach
    • Tworzenie odpowiednich parserow
    • Tworzenie nowych watkow dla parserow (1 spolka == 1 watek)
    • Pobieranie danych
    • Parsowanie danych
    • zapisywanie danych do pliku albo bazy danych
